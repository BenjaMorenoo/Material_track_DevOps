# Track / Taller de DevOps 🚀

Este repositorio contiene el **material complementario** utilizado a lo largo del track (taller) de **DevOps**.

El objetivo del taller es introducir los conceptos fundamentales de DevOps, su cultura, prácticas y herramientas más utilizadas en la industria, combinando teoría con ejercicios prácticos.

---

## 📚 Contenido del repositorio

A lo largo del curso, este repositorio se irá actualizando con:

- Presentaciones utilizadas en clase
- Material teórico de apoyo
- Guías prácticas y ejemplos
- Ejercicios
- Recursos adicionales (documentación, links útiles, etc.)

---

## 🎯 Objetivos del track

- Comprender qué es DevOps y por qué surge
- Conocer la cultura DevOps y sus principios
- Aprender buenas prácticas de automatización
- Introducir conceptos como CI/CD, contenedores e infraestructura como código
- Familiarizarse con herramientas comunes del ecosistema DevOps

---

## 🛠️ Herramientas y tecnologías (introductorio)

Durante el track se trabajará, a nivel introductorio, con herramientas como:

- Git / GitHub
- CI/CD
- Docker
- Conceptos básicos de cloud
- Monitoreo y automatización

*(Las herramientas pueden variar según el avance del taller)*

## 📌 Notas

- Todo el material es de uso educativo.
